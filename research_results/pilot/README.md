# Deep Research Pilot

Generated: 20 SKUs across 3 providers.

## Workflow

1. `claude_pro_batch.json` → Run prompts in Claude.ai (Pro account)
2. `gemini_dr_batch.json` → Run prompts in Google AI Studio
3. `chatgpt_dr_batch.json` → Run prompts in ChatGPT with web search

Each prompt ends with a CRITICAL JSON block — copy/paste the JSON response.

## Result format

Save results as `pilot_results/<pn>_result.json` with the raw JSON block.
