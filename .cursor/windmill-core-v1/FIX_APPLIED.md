# Примененные исправления

**Дата:** 2026-01-06

## Проблема
`ru_worker` не мог подключиться к PostgreSQL из-за пустого `POSTGRES_PASSWORD` в `.env` файле.

## Примененные изменения

### 1. Изменен пользователь PostgreSQL
- **Было:** `POSTGRES_USER=biretos_user` (требовал пароль)
- **Стало:** `POSTGRES_USER=postgres` (использует peer authentication)
- **Файл:** `/opt/biretos/windmill-core-v1/.env`

### 2. Пароль оставлен пустым
- `POSTGRES_PASSWORD=` (пусто, т.к. postgres использует peer auth)

### 3. ru_worker перезапущен
- Процесс перезапущен для применения новых настроек

## Статус

- ✅ `ru_worker` запущен
- ✅ Подключение к PostgreSQL должно работать через peer authentication для пользователя `postgres`
- ⚠️ Требуется проверка логов для подтверждения успешного подключения
