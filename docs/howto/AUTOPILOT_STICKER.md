# AUTOPILOT STICKER

1. В новом чате можно написать: `Автопилот`.
2. В обычном потоке продолжения пишите только: `12`.
3. `12` всегда означает: выполнить ровно одну текущую фазу из `docs/autopilot/STATE.md`.
4. После каждой фазы Cursor обновляет `docs/autopilot/STATE.md` и останавливается.
5. Инструкцию смотрите только в последней строке ответа.
6. Если в последней строке `ВНЕШНИЙ АУДИТ`, нужен внешний JUDGE.
7. При `ВНЕШНИЙ АУДИТ` отправьте JUDGE PACK во внешний чат и дождитесь вердикта.
8. После внешнего вердикта вернитесь в Cursor и снова напишите `12`.
9. Если чат перегружен, откройте новый чат и напишите `Автопилот`.
10. Затем снова пишите `12` для пошагового продолжения.
11. Не пропускайте фазы вручную: автопилот идет строго по одной фазе.
12. Если последняя строка `DONE`, задача закрыта и можно брать следующую.
