# ONE LINER TASK (универсальный вход)

Скопируй шаблон ниже в чат Cursor:

RISK: <LOW|SEMI|CORE>  
TASK: <одна строка>  
FILES: <если известны>  
CONSTRAINTS: <если есть>  
Сначала классифицируй/подтверди риск по `risk_router.mdc`.  
Выдай результат текущей роли (ARCHITECT или CRITIC/PLANNER по необходимости).  
Заверши ответ блоком `WAITING_FOR_APPROVE`.  
Не переходи к следующей фазе без слова `approve`.
