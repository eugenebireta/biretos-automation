# Maintenance Scripts

Резервное место для скриптов сопровождения (фикс багов, массовые апдейты,
ручные патчи). Пока каталог пуст, но создан заранее, чтобы новые утилиты
сразу раскладывать по структуре.

Рекомендации:

- давайте скриптам говорящие имена `fix_*`, `update_*`;
- добавляйте короткий docstring с датой и контекстом задачи;
- после использования переносите устаревшие версии в `scripts/archive/`.
