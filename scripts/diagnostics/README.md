# Diagnostics Scripts

Папка содержит утилиты для проверки качества данных и структуры файлов. Они
используются по мере необходимости для локальных проверок.

| Скрипт | Назначение |
| --- | --- |
| `check_my_top5.py` | Быстрая сверка выбранных TOP-5 SKU в выгрузке |
| `inspect_columns.py` | Проверка структуры колонок в XLSX/CSV |
| `inspect_honeywell.py` | Диагностика текстовых описаний Honeywell |

> При добавлении новых проверок размещайте их здесь, чтобы отличать от
> аналитических задач и черновиков `_scratchpad`.
