## ❌ Проблемы

### 1. Категории не добавляются при создании товара
**Проблема:** Категории добавляются в payload (`payload["categories"] = categories`), но не сохраняются в Shopware при создании через `/api/product`.

**Возможные причины:**
- Shopware требует другой формат для категорий при создании
- Нужно использовать `associations` вместо прямого поля `categories`
- Категории нужно добавлять отдельным запросом после создания товара

**Решение:** Использовать `bulk_update_all_products.py` для добавления категорий к уже созданным товарам.

### 2. categoryId не сохраняется в visibility
**Проблема:** `categoryId` добавляется в payload visibilities при создании товара, но не сохраняется в Shopware. Попытка обновить через `PATCH /api/product-visibility/{id}` также не сохраняет `categoryId`.

**Возможные причины:**
- Shopware 6.7.5.1 не поддерживает `categoryId` в `product_visibility` через API
- Нужно использовать другой способ установки mainCategory
- Возможно, это ограничение версии Shopware

**Попытки решения:**
1. ✅ Добавление `categoryId` в visibilities при создании товара - не работает
2. ✅ Обновление через `PATCH /api/product/{id}` с visibilities - не работает
3. ✅ Обновление через `PATCH /api/product-visibility/{id}` - не работает

**Текущий статус:** `categoryId` не сохраняется в Shopware, несмотря на все попытки.
