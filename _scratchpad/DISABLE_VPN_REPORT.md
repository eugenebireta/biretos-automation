## ИНСТРУКЦИЯ ПО ВОССТАНОВЛЕНИЮ

### Локальный ПК (Windows)

#### Включить Xray:
```powershell
# Запустить Xray процесс
Start-Process "C:\Users\Eugene\xray\xray.exe" -ArgumentList "run", "-config", "config.json"

# Или если есть ярлык в автозапуске:
# Переименовать обратно: Xray.lnk.disabled → Xray.lnk
# в папке: $env:APPDATA\Microsoft\Windows\Start Menu\Programs\Startup
```

#### Включить WireGuard:
```powershell
# Включить службу
Start-Service -Name "WireGuardTunnel$CursorVPN"

# Включить автозапуск
Set-Service -Name "WireGuardTunnel$CursorVPN" -StartupType Automatic
```
