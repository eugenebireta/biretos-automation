# Перепривязка медиа к товарам

## Назначение

Скрипт `rebind_product_media.py` перепривязывает существующие медиа к товарам через Shopware API, чтобы триггернуть генерацию thumbnails.

## Когда использовать

- ✅ Товары уже созданы
- ✅ Медиа уже загружены и корректны
- ✅ `media_type` уже исправлен (`media:generate-media-types` выполнен)
- ❌ Thumbnails для старых товаров не генерируются

## Что делает скрипт

1. Получает все товары из Shopware
2. Для каждого товара получает список медиа (через `product-media` endpoint)
3. Если медиа есть — переотправляет их через `PATCH /api/product/{id}`
4. Сохраняет порядок (position) и не меняет coverId

## Использование

```bash
python _scratchpad/rebind_product_media.py
```

## После выполнения

**ВАЖНО:** Скрипт НЕ вызывает генерацию thumbnails автоматически.

Выполните вручную:
```bash
docker exec shopware php bin/console media:generate-thumbnails
```

## Особенности

- ✅ Идемпотентен — можно запускать многократно
- ✅ Батчинг — обрабатывает по 50 товаров с задержкой
- ✅ Rate limit friendly — небольшие задержки между батчами
- ✅ Обработка ошибок — при ошибке товара продолжает работу
- ✅ Логирование — подробные логи для каждого товара

## Ограничения

- ❌ НЕ загружает файлы заново
- ❌ НЕ создаёт новые media
- ❌ НЕ трогает thumbnail sizes
- ❌ НЕ лезет в БД напрямую
- ✅ Работает ТОЛЬКО через Shopware API

## Ожидаемый результат

После выполнения и генерации thumbnails:
- ✅ Thumbnails появятся на диске
- ✅ Превьюшки появятся в листингах
- ✅ Превьюшки появятся в карточках товаров
- ✅ Превьюшки появятся в админке
