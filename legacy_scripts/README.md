# Legacy Scripts — Обзор

Четыре автономных скрипта, написанных до внедрения основной платформы.
Каждый решает конкретную бизнес-задачу и является кандидатом на миграцию
в соответствующий этап roadmap.

## Детальные заметки

### `wordstat/`
- **Скрипты:** `combine_wordstat.py` (v1), `combine_wordstat_v3.py` (актуальная версия), `run_wordstat.bat`
- **API:** нет — только локальная обработка файлов
- **Библиотеки:** `pandas`, `argparse`, `pathlib`, `re`
- **Особенности:** Обрабатывает как форматы `YYYY-MM`, так и `Май 2023`; умеет определять ключевое слово из колонки или из имени файла.

### `возвраты/`
- **Скрипты:** `объединяем_возвраты.py`, `return.bat`
- **API:** нет
- **Библиотеки:** `pandas`, `glob`, `openpyxl`
- **Особенности:** Пути захардкожены под конкретного пользователя — при миграции нужно параметризовать через аргументы или конфиг.

### `СДЭК api/`
- **Скрипты:** `cdek_create_order.py` (основной), `test_py.py` (прототип с захардкоженными кредами — ⚠️ не деплоить)
- **API:** CDEK REST API v2 (`/oauth/token`, `/location/cities`, `/deliverypoints`, `/orders`), InSales REST API (`/orders.json`, `/clients/{id}.json`, `/orders/{id}/fields/{field_id}.json`), OpenStreetMap Nominatim (геокодирование)
- **Библиотеки:** `requests`, `python-dotenv`, `math`
- **Захардкожено:** тариф 136, точка отгрузки `CHEL172`, поле трека `21837095`, габариты 40×30×10 см — при миграции вынести в конфиг
- **⚠️ Безопасность:** `test_py.py` содержит открытые креды CDEK — файл нужно удалить или добавить в `.gitignore` при миграции

### `данные из инвойса/`
- **Скрипты:** `extract_invoice.py`
- **API:** нет (только `pdfplumber` для локальных PDF)
- **Библиотеки:** `pdfplumber`, `pandas`, `glob`, `re`
- **Особенности:** Позиции колонок [1, 6, 10] захардкожены под конкретный шаблон накладной; если формат изменится — нужна доработка детектора колонок.

## Приоритет миграции

| Приоритет | Папка | Причина |
|---|---|---|
| 🔴 Высокий | `СДЭК api/` | Уже используется в продакшне; есть незащищённые креды; нужен трекинг в Core |
| 🔴 Высокий | `данные из инвойса/` | Блокирует Этап 16 (Procurement); ручной процесс |
| 🟡 Средний | `возвраты/` | Нужен для Этапа 17; сейчас полностью ручной |
| 🟢 Низкий | `wordstat/` | Аналитика, не критичный путь; можно мигрировать в Этап 18 |
